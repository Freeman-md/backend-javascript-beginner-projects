# Backend JavaScript Beginner Projects

This repository contains my beginner backend projects built with JavaScript and Node.js.

The projects come from the Backend Developer roadmap and are organized as standalone folders. Each project has its own README, source code, tests, and implementation notes where needed.

## Projects

- [01-cli-task-tracker](./01-cli-task-tracker) - A command-line task tracker built with Node.js, filesystem storage, and JSON persistence.

## Purpose

The goal is to strengthen backend fundamentals through small, focused projects before moving into larger backend systems.

These projects reinforce:

- Node.js fundamentals
- CLI and API development
- filesystem and database persistence
- validation and error handling
- testing
- documentation
- incremental Git history
